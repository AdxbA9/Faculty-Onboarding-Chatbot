\# MCBV6



\## Overview

MCBV6 is the standalone optimized RAG script version of the chatbot.



\## Main features

\- automatic PDF detection

\- PDF text extraction

\- chunking with overlap

\- embeddings with `all-mpnet-base-v2`

\- FAISS retrieval

\- Cross-Encoder reranking

\- generation with `flan-t5-xl`

\- embedding caching

\- phone and count fallback handling

\- best source and evidence snippet output



\## Improvement focus

Compared with earlier versions, MCBV6 is more practical and reusable as a standalone script rather than a notebook.
