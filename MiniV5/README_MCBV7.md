\# MCBV7



\## Overview

MCBV7 keeps the same RAG retrieval pipeline as MCBV6, but replaces heavy local generation with Groq API generation.



\## Main changes from MCBV6

\- kept local retrieval

\- replaced local `flan-t5-xl` generation with Groq

\- improved speed of answer generation

\- reduced local hardware pressure

\- preserved source citation behavior



\## Improvement focus

The main improvement in MCBV7 is faster and more practical answer generation while preserving the same retrieval architecture.
